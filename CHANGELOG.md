# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-02

### Added
1. Public open-source packaging configuration for `ypt`.
2. Legal-compliant README with NetAcad download requirement.
3. Contributor documentation and project policies.
4. Pytest test suite (unit and integration coverage).
5. GitHub Actions workflow for automated tests.

### Changed
1. Standardized CLI entrypoints to `ypt`.
2. Simplified module runner to delegate to `ypt.cli`.
